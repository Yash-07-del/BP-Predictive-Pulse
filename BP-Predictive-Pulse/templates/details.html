About Predictive Pulse 🩺

Predictive Pulse is a machine learning-powered web application that predicts the stage of blood pressure based on patient data. It helps in early detection and awareness of:

  • 🟢 Normal BP
  • 🟡 Hypertension Stage-1
  • 🟠 Hypertension Stage-2
  • 🔴 Hypertensive Crisis

This app is built using Python, Flask, Pandas, Scikit-learn, and Bootstrap.
